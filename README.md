# 🐁⚖️ MiceSplit App : Body-Weight Based Animal Grouping 🐁⚖️

### MiceSplit App is a lightweight web application for grouping animals into balanced experimental groups based on body weight.  

* It is suitable for mice, rats, or any other animal model where weight matching is required prior to experimental procedures.

* The app supports transparent and reproducible group assignment using simple CSV input files and an intuitive web interface.



### Features ✅

* Upload CSV files containing animal IDs and body weight

* Automatically split animals into balanced groups

* Hierarchical clustering to minimise between-group weight differences

* Interactive web interface with downloadable results
